Design UI/UX

Apa itu Design UI?

User interface (UI) design atau desain antarmuka adalah proses yang digunakan desainer untuk membangun antarmuka dalam perangkat lunak yang berfokus pada penampilan. UI designer bertujuan untuk membuat tampilan antarmuka yang mudah digunakan dan menyenangkan pengguna.

Design UX

Apa itu UX dan contohnya? User Experience (UX)adalah bagaimana seorang pengguna internet mengakses website, suatu pengalaman yang mereka dapatkan dari website tersebut. Contoh User Experience adalah mengeksplorasi semua fitur website yang ada, melihat tampilan website-nya, dan melakukan prosedur hingga berhasil transaksi produk/jasa.

Tau kah kamu kalau website ini tuh ada unsur Design UI namun tidak ada unsur Design UX. itu karena website ini dibuat hanya untuk dilihat-lihat saja, tidak ada tombol untuk bercakap dan berinteraksi.
